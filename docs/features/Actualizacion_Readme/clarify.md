# Clarificaciones: Actualizacion de Readme

No hay clarificaciones adicionales, el requerimiento es unificar la información del proyecto y evitar que se fragmente en dos archivos README.

## Resoluciones
- Los comandos de desarrollo indicarán la necesidad de ingresar a `src` (p. ej., `cd src && npm install`).
- El `README.md` final estará exclusivamente en la raíz del repositorio, eliminando el de `src/`.
- Se reflejará correctamente la arquitectura de Multi Agente GESFER, SddIA, etc.
