# Objetivos: Actualizacion de Readme

## Objetivo Principal
Unificar el archivo `README.md` principal con `src/README.md` y adecuar el contenido para que sea un reflejo preciso del proyecto actual (GesFer Admin Frontend).

## Objetivos Específicos
1. Consolidar la información de ambos `README.md` (raíz y `src/`) en el archivo `README.md` principal del repositorio.
2. Analizar la estructura y contenido del proyecto para asegurar que el `README.md` consolidado refleje correctamente las tecnologías, estructura, configuración y comandos de ejecución.
3. Eliminar el archivo `src/README.md` tras haber movido su información al archivo principal, para mantener una Única Fuente de Verdad (SSOT).
4. Cumplir con la norma de SSOT documentada (no documentos técnicos sueltos, uso correcto de jerarquía).
