# Actualización de Readme

**Propósito:** Consolidar la documentación del proyecto eliminando la redundancia de tener múltiples archivos README.md.

## Alcance
- Archivo raíz `README.md`.
- Archivo `src/README.md`.

## Acciones Relacionadas
- Unión del contenido de ambos archivos, dando prioridad a las instrucciones y estructura general del proyecto.
- Eliminación del archivo de nivel inferior (`src/README.md`) una vez incorporado su contenido.
