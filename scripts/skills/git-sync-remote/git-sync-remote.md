# git-sync-remote

Cápsula: `paths.skillCapsules.git-sync-remote`.

## Request

| Campo | Tipo | Default |
|-------|------|---------|
| `remote` | string | `origin` |
| `fetch` | bool | true |
| `pull` | bool | true |
| `push` | bool | false |
| `branch` | string | rama actual |

## CLI

`Git-Sync-Remote.bat [--remote origin] [--branch x] [--no-fetch] [--no-pull] [--push]`
